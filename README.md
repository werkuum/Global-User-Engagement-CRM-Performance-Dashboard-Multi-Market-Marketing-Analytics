# Global-User-Engagement-CRM-Performance-Dashboard-Multi-Market-Marketing-Analytics
Цей проєкт присвячений створенню комплексної аналітичної системи для відстеження життєвого циклу користувачів та ефективності емейл-маркетингу на глобальному рівні. Мета — об'єднати дані про реєстрації акаунтів та активність у розсилках для визначення ключових ринків і сегментації аудиторії.
